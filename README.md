# 🏛️ Sovereign Strategy | Elite Investment Consultancy

https://gdev-z.github.io/sovereign-strategy-refined/

**Sovereign Strategy** é uma Landing Page de altíssimo padrão desenvolvida para o nicho de consórcios imobiliários e de luxo. O foco do projeto é converter leads qualificados através de uma interface que transmite autoridade, segurança financeira e exclusividade.

---

## 🚀 Tecnologias Utilizadas

* **React + Vite:** Performance e rapidez no carregamento.
* **Tailwind CSS:** Estilização baseada no sistema de design "Sovereign Aesthetic".
* **Lucide React:** Ícones minimalistas e modernos.
* **Framer Motion:** Animações fluidas (Elegant Fade) para uma experiência premium.

---

## 🎨 Design System: The Sovereign Aesthetic

O projeto segue uma estratégia visual baseada no **Tenebrismo Digital**, utilizando contrastes profundos e o dourado Champagne (`#D4AF37`) para guiar o olhar do usuário.

* **Tipografia:** Mix de fontes serifadas (Playfair Display) para luxo e sans-serif (Inter/Manrope) para modernidade.
* **Asimetria Intencional:** Foge dos templates comuns para criar uma sensação de escritório privativo digital.
* **Mobile-First:** Totalmente otimizado para navegação via smartphone.
